Detailed Description

Library of compiled shader modules.

Shader library manages bytecode for native gfx shader modules for each platfrom, which required for actual gfx shader programs construction. Shader library caches new compiled modules, allows to create gfx shaders required, and allows to store and load cache from file system on demand.

In actual game no shader compiler is allowed at runtime. Thus, all gfx shaders creation will be handled using cached bytecode of modules in a shader library for a specific platform.

Note
Thread-safe
