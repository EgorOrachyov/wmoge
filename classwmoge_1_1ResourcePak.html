Detailed Description

Interface for the package of the resources on disc.

ResourcePak abstracts access to the resources on disk. It provides ability to load a particular resource meta file from a resource name, and allows to read a raw data using path.

Internally resource pack can be represented as a wrapper for a file system resource directory, or it can manage a compressed pak of resources on a disk.
