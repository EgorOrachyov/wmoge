Detailed Description

Controls the rendering of the mesh geometry.

Material is composed of the shader object and a set of material params. Material shader object defines the set of available params for rendering settings. Material params provide user the ability to easily set params to tweak rendering of the concrete object. User can apply shader to a given mesh geometry and issuer rendering on a GPU.

Note
Particular shader variation depends on a mesh properties and other settings, thus huge number of materials with different settings may cause a significant increase of shader variations count.
