Detailed Description

Utility-class for rendering debug geometry and text.

Aux draw manager provides a low-level possibility to draw basic primitives, shapes, screen and world text for debug purposes onto final rendered image. Also supports persistent primitives with desired life-time.

Note
thread-safe
