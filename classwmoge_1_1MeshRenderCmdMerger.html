Detailed Description

Performs merging of sorted draw cmds accordingly to assigned bucket slots.

Traverses sorted queues and merges consecutive cmds with the same bucket slot index. When traverses and merges, pathces cmds, adding prmitive ids per-inst vertex attribute to fetch data on the gpu from the per-frame global storage.
