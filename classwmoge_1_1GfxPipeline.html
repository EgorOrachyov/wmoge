Detailed Description

Represents created and compiled graphics pipeline state object.

Pipeline is a compete object which can be directly bound to the command list for the rendering. Pipeline creation depends on a render pass and shader. As soon as dependencies created pipeline creation is started. Pipeline creation is asynchronous and done in the background. When pipeline created it will be used in the rendering.
