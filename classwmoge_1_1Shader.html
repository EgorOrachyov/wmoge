Detailed Description

An interface to the shader class.

Shader provides fundamental interface to the shading algorithms. It provides full information about the the shader, parameters, spaces, techniques, passes, pipeline setup and pre-compilation options.

Shader allows to get or compile a particular variations (permutations) of gpu programs with desierd defines setup. Also it keeps a cache of already compiled programs, and serializes this cahce to load on next run.
