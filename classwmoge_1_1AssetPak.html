Detailed Description

Interface for the package of the assets on disc.

AssetPak abstracts access to the assets on disk. It provides ability to load a particular asset meta file from a asset name, and allows to read a raw data using path.

Internally asset pack can be represented as a wrapper for a file system asset directory, or it can manage a compressed pak of assets on a disk.
