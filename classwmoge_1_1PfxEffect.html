Detailed Description

Complete pfx effect which can be instantiated for emitting.

Effects is a hierarchical collection of different pfx components. Components are updated in a order of parend-child dependency. Events in a parent component can trigger changes in child components, what can be used to make complex pfx effects.

Effect can be instantiated into a particular running instance using pfx emitter class.
