Detailed Description

Compiles collection of mesh batches into set of render commands for rendering.

Compiles mesh batches into a set of render cmds, for each camera and each pass, which pass filtering and relevance options. For batches which supports instancing, compiler assignes bucket slots for further render commands merging.
