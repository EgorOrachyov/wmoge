Detailed Description

Base class for any scene tree object or asset with reflection support.

Object is a thin as possible class which comes with only thread-safe reference counting and single virtual table pointer overhead. The object provides an interface for common methods, which may be required to manipulate objects and write generalized runtime code.

The main feature of the object is the runtime type information support (or reflection). It is possible to get object class, query its inheritance tree, get all methods, properties and fields. Object can be used for more simple automated serialization and deserialization, viewing its structure in the inspector and so on.
