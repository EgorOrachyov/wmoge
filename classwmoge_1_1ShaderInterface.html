Detailed Description

An interface to the shader class.

Shader interface provides fundamental interface to the shading algorithms. It provides full information about the the shader, parameters, spaces, techniques, passes, pipeline setup and pre-compilation options.

Shader interface allows to get or compile a particular variations (permutations) of gpu programs with desierd defines setup. Actual implementation of the interface
