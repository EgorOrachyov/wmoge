Detailed Description

Thread-safe wrapper for gfx context to be used from any thread.

Wraps GfxCtx interface. It uses commands serialization to send it to a separate gfx thread, responsible for GPU communication. Simple commands sent with no wait. Commands requiring immediate feedback require to wait until gfx thread process them.
