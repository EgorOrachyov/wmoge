Detailed Description

Complete gpu program (shader with complete set of stages) for the execution.

Shaders consists of a number of stages for execution. Shader can be created from any engine thread. Shader compilation is asynchronous and done in the background. As soon as shader compiled, it will be used in the rendering pipeline.

Possible shader stages sets:

  • vertex and fragment for classic rendering
  • compute for computational pipeline
