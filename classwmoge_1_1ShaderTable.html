Detailed Description

Table with common render system shaders.

Shader table provides access to built-in engine shaders, declared in engine shaders folder. This shader table shaders are a type-safe compile-time wrapper for engine shaders. It can be used to reduce boilerplate in case of writing engine related rendering code.
