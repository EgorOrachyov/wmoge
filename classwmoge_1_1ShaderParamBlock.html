Detailed Description

Holds params set which can be bound to a shader for drawing.

ShaderParamBlock is a smart and comfort abstraction over raw gfx descriptor set management, required for dispatching shader on a gpu. Prefere its usage instead of raw gfx descriptor sets.

ShaderParamBlock allows to set textures, buffers, and raw params using param id and set methods. Data automatically packed internally into proper places. Default values setup automatically as well, based on shader class defenition.

ShaderParamBlock automatically creates and update buffers and descriptor sets on a gpu. After validation it can be directly inserted into desired descriptor set slot for drawing.
