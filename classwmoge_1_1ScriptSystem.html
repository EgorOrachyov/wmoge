Detailed Description

Engine script system responsible for scripting features for a game.

Scripting system allows to attach scripts to any engine objects to extend their logic, get notifications on events, create objects from scripting, manage scene, process events, updates, etc.

Scripting system takes responsibility on managing objects lifetime, scripts and class loading, global states, and error tracking.

From a user perspective it is enough to create script and attach it to a scene object. The engine will do the rest.
