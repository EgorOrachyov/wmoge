Detailed Description

Some parallel-for job to operate on a set of elements.

This is special type of a task. In fact, it may spawn a set of jobs to be executed in a task manager. This jobs will share elements to operate, using work-stealing principle. Each job is responsible for a batch of a work.

Typical use cases:

  • Run a parallel job for a set of equal objects
  • Background tasks using worker threads for a heavy objects array processing

How to use:

  • Create task, specifying runnable function, name and manager
  • Schedule task to run, specifying number of objects and batch size
