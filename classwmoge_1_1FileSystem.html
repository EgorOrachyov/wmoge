Detailed Description

Abstracts access to the engine and game files directory.

File system abstracts the way how engine files are stored on a target machine. It uses '/' as a universal delimiter and directory separator. It provides a domain prefix in a form of <PREFIX>://<PATH> to specify paths. Standards prefixes are root://, res:// and cache://. This prefixes must be used for all paths to access engine files.

Prefixes description:

  • root:// prefix to a file relative to the engine directory
  • res:// prefix relative to a engine resources directory
  • cache:// prefix relative to engine cache directory for a cached files
