Detailed Description

Reprsents a particular shader program class.

GrcShaderClass is a high level representation of a shading program. It provides a connection between raw glsl sources code of a shader, material and engine gfx module for runtime usage.

GrcShaderClass provides info about a particular shader type. It provides layout information, parameters and structures layout, defines and compilations options, constants and includes, and provides hot-reloading mechanism for debugging.

GrcShaderClass is a template shader for drawing with pre-defined interface. It is not suitable for rendering. In order to get a concrete instance of compiled gpu program, pass and options must be provided from GrcShader.
