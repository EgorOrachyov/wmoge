Detailed Description

Sprite is a sequence of images in a texture which can be animated.

Sprite is a single of a number of images which can be used to draw and animate 2d object.

Sprite consists of a set of animations. Each animation can have unique texture atlas with frames. Frames in the animation describe, which part of the atlas must rendered.

An animation can have a speed (number of frames shown per second) and loop (true if animation must be played in cycled fashion). Sprite have a size in canvas units. An animation can be played by its name.
