Detailed Description

A prefab resource which can be instantiated into a set of objects.

Prefab stores a sub-tree of scene nodes, which can be instantiated and added to a scene tree at once. Prefab allows to make a complex object composed from nodes and use it instantiate multiple times and keep all instances in sync.
