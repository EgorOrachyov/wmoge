Detailed Description

Base class for any gfx resource object.

Base class for any gfx resource, created by driver and exposed to the user. Uses automated reference counting, required for safe resource life-time tracking and passing among several system threads.

Resource utilise gfx thread callback queue, so its creation/destruction is deferred and executed only on gfx thread as a command.
