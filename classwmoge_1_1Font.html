Detailed Description

Font resource for rendering texts from true type fonts.

Font resources stores gpu bitmaps with font glyphs data as well as glyph metrics per each loaded font symbol. Font allows to render text using multiple sizes. Allow to compute precis text strings sizes using selected font size settings.
